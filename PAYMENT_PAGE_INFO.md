# Страница оплаты - Инструкция

## Что добавлено

### Отдельная страница оплаты (payment.html)
При нажатии на любую кнопку покупки (донат, рилики, размут, разбан) пользователь переходит на отдельную страницу оформления заказа.

## Структура страницы

### 1. Информация о покупке
Отображает что покупается и стоимость:
- Привилегия + название (HERO, TITAN и т.д.)
- Количество риликов
- Услуга (размут/разбан)

### 2. Форма никнейма
- Поле для ввода игрового никнейма
- Максимум 16 символов (как в Minecraft)
- Подсказка "Укажите точный никнейм, как в игре"

### 3. Способы оплаты

**Банковская карта:**
- VISA
- MasterCard
- МИР
- (пока недоступна, показывает alert)

**Перевод на карту:**
- Карта МИР для РФ: **5394 2801 5424 7473**
- Резервная карта для Украины: **4874 0700 7193 6866**
- Кнопки "Скопировать" для быстрого копирования

### 4. Инструкция по оплате
1. Скопируйте номер карты
2. Переведите указанную сумму
3. Нажмите "Я оплатил"
4. Свяжитесь с администрацией для подтверждения

### 5. Кнопка "Я оплатил"
После нажатия:
- Показывает благодарность
- Просит связаться с администрацией
- Возвращает на главную через 2 секунды

## Изменения в риликах

### Старое
- 1₽ = 2 рилика
- Максимум: 20 000 риликов

### Новое
- **1₽ = 5 риликов**
- Максимум: 100 000 риликов
- Минимум: 5 риликов

## Улучшенный ползунок

### Старый дизайн
- Простой ползунок
- Количество в тексте
- Минимальный стиль

### Новый дизайн
- **Красивый градиентный ползунок** (голубой)
- **Большая круглая ручка** с свечением
- **Квадратик с количеством** под ползунком:
  - Голубая рамка с свечением
  - Большой шрифт для числа риликов
  - Подпись "риликов"
  - Эффект увеличения при наведении
- Плавные анимации

## Логотип

### Расположение
- Перед названием "DELTAWORLD" в навигации
- Размер: 32x32 пикселя
- Автоматически скрывается если файл не найден

### Файл
Положите файл `logo.png` в папку `public/`

Если файла нет - ничего страшного, название будет без логотипа.

## Параметры URL

Страница payment.html принимает параметры:

### Для привилегий:
```
payment.html?type=rank&item=hero
```

### Для риликов:
```
payment.html?type=relics&amount=1000&price=200
```

### Для размута:
```
payment.html?type=unmute
```

### Для разбана:
```
payment.html?type=unban
```

## Файлы

- `public/payment.html` - страница оплаты
- `public/payment.css` - стили страницы
- `public/payment.js` - логика страницы

## Тестирование

1. Откройте главную страницу
2. Нажмите на любую кнопку покупки
3. Проверьте отображение информации
4. Введите никнейм
5. Выберите способ оплаты "Перевод на карту"
6. Проверьте копирование номеров карт
7. Нажмите "Я оплатил"
