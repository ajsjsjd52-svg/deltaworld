# Быстрый старт Deltaworld Website

## 1. Подготовка (5 минут)

### Скачайте и установите Node.js
- Перейдите на https://nodejs.org/
- Скачайте LTS версию
- Установите с настройками по умолчанию

### Проверьте установку
Откройте командную строку (cmd) и выполните:
```bash
node --version
npm --version
```

## 2. Настройка сервера (3 минуты)

### Откройте файл `.env` в текстовом редакторе

Измените настройки под ваш сервер:

```env
PORT=3000                          # Порт веб-сервера (можно оставить)
RCON_HOST=127.0.0.1               # IP вашего MC сервера (127.0.0.1 если на этом же ПК)
RCON_PORT=25575                    # RCON порт (по умолчанию 25575)
RCON_PASSWORD=ваш_пароль_rcon     # ВАШ RCON ПАРОЛЬ
MC_SERVER_IP=play.deltaworld.ru    # Адрес для отображения на сайте
MC_SERVER_PORT=25565               # Игровой порт сервера
```

### Настройте RCON на Minecraft сервере

Откройте `server.properties` вашего Minecraft сервера и добавьте/измените:

```properties
enable-rcon=true
rcon.port=25575
rcon.password=ваш_пароль_rcon
```

**Перезапустите Minecraft сервер!**

## 3. Запуск (1 минута)

### В папке с проектом выполните:

```bash
npm start
```

### Откройте браузер:

```
http://localhost:3000
```

## 4. Проверка работы

✅ Проверьте что:
- Главная страница открывается
- Отображается IP сервера
- Показывается количество игроков онлайн
- RCON консоль принимает пароль
- Команды выполняются

## Первые команды для теста

В RCON консоли попробуйте:
```
list
time set day
say Привет из веб-консоли!
```

## Частые проблемы

### "Не удалось подключиться к серверу"
- ✓ Проверьте, что Minecraft сервер запущен
- ✓ Проверьте настройки RCON в server.properties
- ✓ Перезапустите Minecraft сервер после изменения RCON настроек
- ✓ Проверьте что RCON_HOST и RCON_PORT правильные в .env

### "Неверный пароль" в RCON
- ✓ Убедитесь что пароль в .env совпадает с паролем в server.properties
- ✓ Проверьте что нет лишних пробелов в пароле

### "Не показывает игроков"
- ✓ Проверьте что кто-то действительно онлайн на сервере
- ✓ Убедитесь что RCON правильно настроен
- ✓ Проверьте консоль браузера на ошибки (F12)

### Firewall блокирует RCON
- ✓ Откройте порт 25575 в Windows Firewall
- ✓ Если сервер на другой машине - проверьте сетевые настройки

## Кастомизация

### Изменить название сервера
Откройте `public/index.html` и найдите "DELTAWORLD" - замените на свое название

### Изменить цвета
Откройте `public/styles.css` и измените переменные в начале:
```css
:root {
    --primary: #00d9ff;      /* Основной цвет */
    --secondary: #7b2ff7;    /* Дополнительный */
    --accent: #ff006e;       /* Акцент */
}
```

### Изменить особенности сервера
В `public/index.html` найдите секцию "Особенности сервера" и измените текст
