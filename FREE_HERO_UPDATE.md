# Обновление: Бесплатная привилегия HERO

## Что изменилось

Привилегия **HERO** теперь **полностью бесплатная** (0 звезд ⭐).

## Как это работает

### В Telegram боте:
1. Пользователь выбирает "Донат привилегии" → "🎁 HERO (БЕСПЛАТНО)"
2. Вводит свой игровой никнейм
3. **Привилегия выдается СРАЗУ** через RCON без оплаты
4. Получает сообщение: "✅ Привилегия HERO успешно выдана! Привилегия активирована бесплатно! 🎁"

### На сайте:
- Карточка HERO показывает: "🎁 БЕСПЛАТНО"
- Кнопка изменена с "Купить" на "Получить"

### При переходе с сайта:
- Если пользователь нажимает "Получить" на HERO → переходит в бота
- Бот сразу выдает привилегию после ввода никнейма (без запроса оплаты)

## Технические изменения

### Файлы изменены:
1. **config/ranks.json** - цена HERO: `0` звезд
2. **server.js** - логика для бесплатной выдачи без инвойса
3. **api/index.js** - логика для webhook режима
4. **public/index.html** - отображение "🎁 БЕСПЛАТНО"

### Логика выдачи:
```javascript
// Если привилегия стоит 0 звезд
if (rank.stars === 0) {
    // Сразу выполняем RCON команду
    await executeRconCommand(command);
    // Отправляем подтверждение
    bot.sendMessage("✅ Привилегия выдана бесплатно!");
}
// Для платных привилегий - создаем инвойс
else {
    await bot.sendInvoice(...);
}
```

## Преимущества

✅ **Нет технических ограничений** - Telegram Stars не поддерживает 0 звезд, поэтому бесплатная привилегия выдается напрямую  
✅ **Мгновенная выдача** - без создания инвойса  
✅ **Привлечение новых игроков** - бесплатная стартовая привилегия  
✅ **Тестирование системы** - игроки могут попробовать донат-систему бесплатно  

## Деплой

Изменения загружены на GitHub:
```bash
git commit -m "Make HERO rank free (0 stars)"
git push origin main
```

На Railway проект обновится автоматически (если включен Auto-Deploy).

## Как вернуть платную HERO

Если нужно вернуть HERO платным:
1. Откройте `config/ranks.json`
2. Измените `"stars": 0` обратно на `"stars": 9`
3. Измените `"price": 0` обратно на `"price": 9`
4. Перезапустите сервер
