## Структура сайта

1. **Главная** - IP, статус, количество игроков
2. **Донат** - 9 привилегий с кнопками покупки
3. **Статус** - детальная информация о сервере

## Как работает донат

1. Пользователь нажимает "Купить" на привилегии
2. Вводит свой игровой никнейм
3. Переходит на оплату (настраивается)
4. После оплаты платежная система отправляет POST запрос на `/api/grant-privilege`
5. Сервер автоматически выдает привилегию через RCON

## Настройка выдачи привилегий

В `server.js` найдите:

```javascript
const command = `lp user ${nickname} parent set ${rank}`;
```

Измените команду под ваш плагин прав:
- **LuckPerms**: `lp user <ник> parent set <группа>`
- **PermissionsEx**: `pex user <ник> group set <группа>`
- **GroupManager**: `manuadd <ник> <группа>`

## Создайте группы на сервере

Создайте группы для каждой привилегии:

**LuckPerms:**
```
/lp creategroup hero
/lp creategroup titan
/lp creategroup avenger
/lp creategroup overlord
/lp creategroup minister
/lp creategroup imperator
/lp creategroup daemon
/lp creategroup cobra
/lp creategroup god
```

Настройте права для каждой группы через config файлы LuckPerms.

## Интеграция платежной системы

Смотрите `PAYMENT_SETUP.md` для настройки:
- Юkassa / Robokassa (официальные платежи)
- Donate.stream / Easy Pay (для стримеров)
- Telegram Bot (простой вариант)

После успешной оплаты вызывайте:

```javascript
fetch('https://ваш-сайт.ru/api/grant-privilege', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        nickname: 'Player123',
        rank: 'hero'
    })
});
```

## Безопасность

В продакшене:
1. Добавьте проверку подписи webhook от платежной системы
2. Логируйте все транзакции
3. Используйте HTTPS
4. Храните RCON пароль в .env (не коммитьте!)
