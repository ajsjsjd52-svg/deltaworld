# Финальный дизайн - Северное сияние

## Фон

### Эффект северного сияния (Aurora Borealis)
Создан с помощью двух слоёв анимированных радиальных градиентов:

**Первый слой (body::before):**
- Голубовато-зелёные оттенки
- Цвета: бирюзовый, синий, фиолетовый
- Анимация 20 секунд
- Плавное движение вверх-вниз с изменением масштаба

**Второй слой (body::after):**
- Дополнительные переливы
- Зелёно-синие оттенки
- Анимация 25 секунд (в обратном направлении)
- Создаёт эффект глубины

**Результат:**
Плавно движущиеся световые волны, имитирующие настоящее северное сияние на тёмно-синем фоне.

## Интерактивность

### Карточки доната
**При наведении мышкой:**
- Увеличение на 8% (`scale(1.08)`)
- Подъём на 10px вверх
- Плавная анимация (0.4s с эффектом "пружины")
- Голубое свечение вокруг карточки
- Изменение цвета рамки на голубой

**Featured карточки (MINISTER, GOD):**
- Ещё большее увеличение - 12% (`scale(1.12)`)
- Подъём на 15px
- Более яркое свечение
- Уже имеют начальное свечение

### Карточки услуг (рилики, размут, разбан)
**При наведении:**
- Увеличение на 8%
- Подъём на 10px
- Такая же плавная анимация
- Голубое свечение

## Цвета

### Палитра северного сияния
- **Фон**: Тёмно-синий #0a1628
- **Сияние 1**: Бирюзовый rgba(0, 255, 200, 0.3)
- **Сияние 2**: Голубой rgba(100, 150, 255, 0.3)
- **Сияние 3**: Фиолетовый rgba(150, 100, 255, 0.25)
- **Сияние 4**: Синий rgba(0, 200, 255, 0.25)
- **Сияние 5**: Зелёный rgba(0, 255, 150, 0.2)

### UI элементы
- **Основной**: #4a9eff (голубой)
- **Текст**: #e8e8e8 (светло-серый)
- **Текст вторичный**: #b0c4de (светло-голубой)
- **Границы**: rgba(74, 144, 226, 0.3) (полупрозрачный голубой)

## Эффекты

### Backdrop blur
Все элементы имеют эффект размытия фона:
- Навигация: `backdrop-filter: blur(10px)`
- Карточки: `backdrop-filter: blur(5px)`
- Создаёт эффект "матового стекла"

### Прозрачность
Все элементы интерфейса полупрозрачные, сквозь них виден фон с северным сиянием.

### Тени
При наведении на карточки появляется голубое свечение:
```css
box-shadow: 0 20px 40px rgba(74, 158, 255, 0.4);
```

## Анимация

### Aurora (Северное сияние)
```
0% → 25% → 50% → 75% → 100%
Движение вверх с изменением масштаба и прозрачности
Создаёт эффект волн света
```

### Hover карточек
```css
transition: all 0.4s cubic-bezier(0.175, 0.885, 0.32, 1.275);
```
Плавная анимация с эффектом "пружины" (bounce)

## Технические детали

### Производительность
- Используются CSS трансформации (transform) - аппаратное ускорение
- Анимация на псевдоэлементах (::before, ::after)
- Не нагружает основной поток

### Совместимость
- Работает во всех современных браузерах
- `backdrop-filter` поддерживается в Chrome, Safari, Edge
- Graceful degradation для старых браузеров

Красивый динамичный дизайн с эффектом северного сияния:
- ✨ Плавно движущиеся световые волны на фоне
- 🎯 Карточки увеличиваются и "всплывают" при наведении
- 💎 Полупрозрачные элементы с размытием
- 🌊 Ощущение глубины и объёма
- 🎨 Приятная голубовато-зелёная палитра
