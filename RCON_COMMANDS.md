# Популярные RCON команды для Minecraft

## Основные команды

### Информация о сервере
```
list                    # Список игроков онлайн
tps                     # TPS сервера (требует Spigot/Paper)
version                 # Версия сервера
help                    # Список всех команд
```

### Управление игроками
```
kick <игрок> [причина]           # Выгнать игрока
ban <игрок> [причина]            # Забанить игрока
ban-ip <IP> [причина]            # Забанить IP
pardon <игрок>                   # Разбанить игрока
pardon-ip <IP>                   # Разбанить IP
whitelist add <игрок>            # Добавить в белый список
whitelist remove <игрок>         # Удалить из белого списка
op <игрок>                       # Выдать права оператора
deop <игрок>                     # Забрать права оператора
```

### Сообщения
```
say <сообщение>                  # Отправить сообщение от сервера
tell <игрок> <сообщение>         # Личное сообщение игроку
```

### Управление миром
```
time set day                     # Установить день
time set night                   # Установить ночь
weather clear                    # Ясная погода
weather rain                     # Дождь
weather thunder                  # Гроза
difficulty peaceful              # Мирный режим
difficulty easy                  # Легкий режим
difficulty normal                # Нормальный режим
difficulty hard                  # Сложный режим
gamerule doDaylightCycle false   # Остановить время
```

### Телепортация
```
tp <игрок1> <игрок2>                    # Телепортировать игрока1 к игроку2
tp <игрок> <x> <y> <z>                 # Телепортировать на координаты
```

### Предметы
```
give <игрок> <предмет> [количество]     # Выдать предмет
clear <игрок>                           # Очистить инвентарь
```

### Эффекты
```
effect give <игрок> <эффект> [длительность] [сила]
effect clear <игрок>                    # Снять все эффекты
```

## Команды для управления сервером

```
save-all                        # Сохранить мир
save-off                        # Отключить автосохранение
save-on                         # Включить автосохранение
stop                           # Остановить сервер (осторожно!)
```

## Spigot/Paper команды

```
tps                            # Показать TPS
timings                        # Профилирование производительности
plugins                        # Список плагинов
reload                         # Перезагрузить конфигурацию (не рекомендуется)
```

## Примеры использования

### Приветствие нового игрока
```
say Добро пожаловать на Deltaworld, <игрок>!
give <игрок> diamond 5
```

### Событие на сервере
```
say [СОБЫТИЕ] Через 5 минут начнется PvP турнир!
time set 0
weather clear
```

### Очистка лагов
```
save-all
lagg clear
```

### Бан нарушителя
```
ban <игрок> Использование читов
say Игрок <игрок> был забанен за читерство
```

## Безопасность

⚠️ **ВАЖНО**: 
- Не используйте команду `stop` через веб-консоль без необходимости
- Будьте осторожны с командой `op` - это полные права
- Всегда указывайте причину при бане
- Регулярно проверяйте список операторов

## Полезные плагины для RCON

1. **EssentialsX** - расширенные команды
2. **WorldEdit** - редактирование мира
3. **CoreProtect** - логирование действий
4. **LuckPerms** - управление правами
