# Deltaworld Website - Информация о проекте

## 📋 Описание проекта

Современный веб-сайт для Minecraft сервера **Deltaworld** с функциями:
- 🎮 Отображение статуса сервера в реальном времени
- 👥 Мониторинг игроков онлайн
- 🎛️ RCON консоль для управления сервером
- 📊 Информация о TPS и производительности
- 🌙 Темный дизайн с плавными анимациями
- 📱 Адаптивная верстка для всех устройств

## 📡 API Endpoints

### GET `/api/server-status`
Получает информацию о статусе сервера

**Ответ:**
```json
{
  "online": true,
  "players": "There are 5 of a max of 100 players online: Player1, Player2...",
  "tps": "20.0",
  "ip": "play.deltaworld.ru",
  "port": "25565"
}
```

### POST `/api/rcon-command`
Выполняет RCON команду на сервере

**Запрос:**
```json
{
  "command": "say Hello from web!",
  "password": "rcon_password"
}
```

**Ответ:**
```json
{
  "success": true,
  "response": "Command executed"
}
```

## 🔒 Безопасность

### ⚠️ ВАЖНО для продакшена:

1. **Не используйте простую аутентификацию по паролю**
   - Внедрите JWT токены
   - Используйте сессии
   - Добавьте rate limiting

2. **Настройте HTTPS**
   - Используйте Let's Encrypt
   - Перенаправляйте HTTP на HTTPS

3. **Ограничьте доступ**
   - IP whitelist для RCON
   - Базовая HTTP аутентификация
   - Firewall правила

4. **Защита от атак**
   ```bash
   npm install helmet express-rate-limit
   ```
