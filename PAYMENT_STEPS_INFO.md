# Поэтапная система оплаты - Документация

## Концепция

Система разделена на 3 этапа для удобства пользователя:

### Этап 1: Ввод никнейма (Белый фон)
- Чистый белый экран
- Логотип и название сервера
- Информация о покупке
- Поле для ввода никнейма
- Кнопка "Продолжить"

### Этап 2: Выбор способа оплаты (Темный фон с северным сиянием)
- 5 карточек с методами оплаты
- Каждая карточка - отдельный способ
- При наведении - увеличение
- Цена на каждой карточке

### Этап 3: Детали оплаты (Темный фон)
- Инструкции для выбранного метода
- Реквизиты (для перевода)
- Ссылка на оплату (для онлайн)
- Кнопка "Я оплатил"
- Возможность вернуться назад

## Способы оплаты

### 1. VISA
- **Цвет**: Синий (#1434CB)
- **Тип**: Онлайн оплата
- **Статус**: В разработке
- **Описание**: Перенаправление на платежный шлюз

### 2. MasterCard
- **Цвет**: Красный (#EB001B)
- **Тип**: Онлайн оплата
- **Статус**: В разработке
- **Описание**: Перенаправление на платежный шлюз

### 3. МИР
- **Цвет**: Зелёный (#4DB45E)
- **Тип**: Онлайн оплата
- **Статус**: В разработке
- **Описание**: Перенаправление на платежный шлюз

### 4. Перевод на карту
- **Цвет**: Фиолетовый градиент
- **Тип**: Ручной перевод
- **Статус**: Работает
- **Карты**:
  - РФ (МИР): `5394 2801 5424 7473`
  - Украина: `4874 0700 7193 6866`
- **Инструкция**:
  1. Скопировать номер карты
  2. Перевести сумму
  3. Нажать "Я оплатил"
  4. Связаться с админом

### 5. Telegram Stars
- **Цвет**: Голубой (#0088cc)
- **Тип**: Через Telegram бота
- **Статус**: Требует настройки бота
- **Описание**: Оплата звездами Telegram

## Настройка онлайн оплаты

### Для VISA/MasterCard/МИР

В файле `payment.js` найдите функцию `processOnlinePayment()` и добавьте интеграцию:

```javascript
function processOnlinePayment(method) {
    // Пример для ЮKassa
    const paymentData = {
        amount: purchasePrice,
        currency: 'RUB',
        description: `${type} - ${item || amount}`,
        metadata: {
            nickname: nickname,
            type: type,
            item: item
        }
    };
    
    // Создаём платеж через API
    fetch('/api/create-payment', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(paymentData)
    })
    .then(response => response.json())
    .then(data => {
        // Перенаправляем на страницу оплаты
        window.location.href = data.payment_url;
    });
    
    return false;
}
```

### Для Telegram Stars

1. Создайте Telegram бота через @BotFather
2. Настройте Telegram Stars оплату
3. Измените ссылку в `showPaymentDetails()`:

```javascript
https://t.me/YOUR_BOT_USERNAME?start=pay_${type}_${item || amount}_${nickname}
```

4. Бот должен обрабатывать команду `/start` и создавать invoice для Stars

## Дизайн

### Этап 1 (Белый)
- Минималистичный
- Фокус на вводе никнейма
- Нет отвлекающих элементов

### Этапы 2-3 (Темный с сиянием)
- Эффект северного сияния
- Анимированный фон
- Полупрозрачные карточки
- Эффект "матового стекла"

### Карточки оплаты
- При наведении: увеличение + подъем
- Плавная анимация с "пружинящим" эффектом
- Цветные заголовки по типу карты
- Цена крупным шрифтом

## Логика переходов

```
Этап 1 (Ввод ника)
    ↓ [Продолжить]
Этап 2 (Выбор метода)
    ↓ [Клик на карточку]
Этап 3 (Детали)
    ↓ [Я оплатил]
Главная страница
```

**Возврат:**
- Этап 1 ← [Назад] ← Этап 2
- Этап 2 ← [Назад] ← Этап 3

## Валидация

### Никнейм
- Минимум 3 символа
- Максимум 16 символов
- Обязательное поле

### Цена
- Автоматически рассчитывается
- Отображается на всех этапах
- Одинакова для всех методов

## Интеграция с backend

После оплаты отправляется:

```json
{
  "nickname": "Player123",
  "type": "rank|relics|unmute|unban",
  "item": "hero",
  "amount": "1000",
  "method": "visa|mastercard|mir|transfer|telegram",
  "price": 199,
  "timestamp": "2024-01-01T00:00:00.000Z"
}
```

Это можно использовать для:
- Логирования покупок
- Webhook уведомлений
- Автоматической выдачи после подтверждения
