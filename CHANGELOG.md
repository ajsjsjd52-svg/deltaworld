## Основные изменения

### Дизайн
- Минималистичный темный дизайн без лишних элементов
- Убраны все эмодзи и смайлики
- Упрощены анимации (меньше частиц, медленнее движение)
- Убраны лишние эффекты свечения
- Более простые формы (меньшие border-radius)

- Избыточные анимации

- Выполнение команд через веб-консоль
- Готовая система для интеграции платежей

## Файлы проекта

### Основные
- `server.js` - Backend сервер с Express и RCON
- `public/index.html` - HTML страница
- `public/styles.css` - Минималистичные стили
- `public/script.js` - JavaScript логика
- `.env` - Конфигурация сервера

### Документация
- `START.md` - Быстрый запуск (НАЧНИТЕ ОТСЮДА)
- `README.md` - Основная документация
- `PAYMENT_SETUP.md` - Настройка платежей
- `DEPLOYMENT.md` - Развертывание на VPS
- `RCON_COMMANDS.md` - Список команд

## Следующие шаги

1. Настройте `.env` файл с вашими данными RCON
2. Запустите сервер: `npm start`
3. Настройте интеграцию с платежной системой в `public/script.js`
4. Измените цены и привилегии под ваш сервер
5. Разверните на хостинг (см. DEPLOYMENT.md)

## Кастомизация

### Изменить цвета
Откройте `public/styles.css` и измените переменные:
```css
:root {
    --primary: #00d9ff;    /* Основной цвет */
    --secondary: #7b2ff7;  /* Вторичный */
    --accent: #ff006e;     /* Акцент */
}
```

### Изменить привилегии
Откройте `public/index.html` и найдите секцию donate

### Добавить платежную систему
Смотрите `PAYMENT_SETUP.md`
